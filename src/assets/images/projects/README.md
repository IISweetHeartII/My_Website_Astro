# Project Thumbnails Guide

이 폴더에 프로젝트 썸네일 이미지를 저장하세요.

## 이미지 규격

### 권장 비율: 16:9

- **1920x1080** (Full HD, 추천)
- **1280x720** (HD)
- **1600x900** (중간 크기)

### 파일 형식

- PNG (투명 배경 가능)
- JPG (파일 크기 작음)
- WebP (최적화, 권장)

## 파일명 예시

```
mylunch.png           - 내 점심 뭐 먹지
omechu.png            - 오메추
testopi.png           - Testopi
mathfigure.png        - Math Figure
starterm.png          - Star Term

fortune-meme.png      - 밈 운세 생성기
mockly.png            - Mockly
planner.png           - Smart Planner
```

## 캡처 팁

1. **화면 전체보다는 핵심 UI 중심으로 캡처**
2. **브라우저 개발자 도구로 정확한 비율 맞추기**
3. **밝은 배경에서 캡처 (가독성 향상)**
4. **중요한 기능이 보이도록 구도 잡기**

## 이미지 최적화 (선택사항)

- [TinyPNG](https://tinypng.com/) - PNG 압축
- [Squoosh](https://squoosh.app/) - 다양한 형식 변환 및 압축
- [ImageOptim](https://imageoptim.com/) - 맥용 이미지 최적화 도구
