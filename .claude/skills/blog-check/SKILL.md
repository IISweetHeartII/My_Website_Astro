# blog-check 스킬

블로그 글이 발행 준비가 됐는지 점검하고 문제점을 리포트합니다.

## 실행 순서

1. 인자로 파일 경로가 주어지면 해당 파일을 검사한다
2. 인자가 없으면 `src/content/blog/` 에서 가장 최신 파일을 자동 선택한다
3. 아래 체크리스트를 모두 점검한다
4. 결과를 표 형식으로 출력하고, 문제가 있으면 수정 방법을 함께 제안한다

## 체크리스트

### 필수 Frontmatter

| 항목 | 조건 |
|------|------|
| `title` | 존재, 60자 이내 |
| `description` | 존재, 150자 이내 |
| `publish` | 존재 (true/false 확인) |
| `created_date` | 존재, YYYY-MM-DD 형식 |
| `slug` | 존재, 영문 소문자 + 하이픈만 사용 |
| `tags` | 1개 이상 존재 |
| `category` | 개발/회고/교육/생산성/일상 중 하나 |

### SEO

| 항목 | 조건 |
|------|------|
| `meta_title` | 존재, "제목 \| 김덕환" 형식 권장 |
| `meta_description` | 존재, 120-160자 권장 |
| `og_title` | 존재 |
| `og_description` | 존재 |
| `og_type` | `article` 이어야 함 |

### 이미지

| 항목 | 조건 |
|------|------|
| `featured_image` | `/images/blogs/[번호]/[번호]_00_thumbnail.png` 형식 권장 |
| `featured_image_alt` | 존재, 비어있지 않음 |
| 본문 이미지 | `![...](/images/blogs/[번호]/[번호]_[순번]_[설명].png)` 형식 사용 여부 확인 |
| 미완성 이미지 | 프롬프트 주석(`<!-- 📸 이미지 프롬프트 -->`)이 아직 남아있으면 경고 |

### 글 품질

| 항목 | 조건 |
|------|------|
| 본문 길이 | 800자 이상 권장 |
| 소제목 | `##` 헤딩 2개 이상 |
| 오프닝 | 첫 문단이 3문장 이내로 짧은지 확인 |

## 출력 형식

```
📋 블로그 글 점검 결과: [파일명]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✅ 통과 항목
- title: "..." (32자)
- description: "..." (98자)
- ...

⚠️  경고 (발행은 가능하나 개선 권장)
- meta_description이 160자를 초과합니다 (현재: 172자)
- ...

❌ 필수 수정 (발행 전 반드시 수정)
- slug가 없습니다
- ...

📊 발행 준비 상태: [준비됨 / 수정 필요 / 미완성]
```

## 발행 준비 상태 기준

- **준비됨**: 필수 항목 모두 통과, `publish: true`
- **수정 필요**: 필수 항목 통과, `publish: false` 또는 경고 항목 있음
- **미완성**: 필수 항목 중 하나라도 누락 또는 이미지 프롬프트 주석 남아있음
